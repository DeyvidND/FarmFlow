# Помощни снимки (Help screenshots)

Сложи реални screenshot файлове тук и те автоматично ще се покажат в екран „Помощ".
Докато няма файл, на негово място стои сив placeholder с името на файла.

Очаквани файлове (PNG или JPG, преименувай на точно тези имена):

| Файл                      | Какво да снимаш                                                        |
| ------------------------- | ----------------------------------------------------------------------|
| `econt-register.png`      | Публичната страница за регистрация на login.econt.com/register        |
| `speedy-api-user.png`     | Имейлът/отговорът от Speedy с API потребител и парола (замажи паролата!) |
| `import-flow.png`         | Наш екран „Внос" с таблицата след качване на файл (логнат в панела)    |

`econt-register.png` и `import-flow.png` (публична страница + наш екран) могат да се
хванат автоматично през browser preview. `speedy-api-user.png` идва само от Speedy по
имейл — задължително ръчно, никога не пращай истинска парола в снимка без замазване.

Препоръка: ширина ~1000px, изрежи само важната част. Файловете се сервират от `/help/<име>`.
