# Доставка с Еконт — как се настройва

Еконт е по желание. Ако доставяш само сам или клиентите идват да си вземат поръчката от
пазара, не ти трябва Еконт — прескочи това.

Има **два начина** да предлагаш Еконт. Избираш ги от **Доставка → Еконт → „Как ще изпращаш с Еконт?“**

---

## 1. Ръчно (най-лесно — без свързване на акаунт)

Клиентът избира Еконт на касата и плаща **фиксирана цена**, която ти задаваш. Поръчката идва
при теб, а **всяка сутрин получаваш имейл** със списък какво да изпратиш днес. Носиш пратките
до Еконт сам (или поръчваш куриер от Еконт както обикновено) и готово.

**Какво ти трябва:** само обикновен Еконт акаунт (безплатен на [econt.com](https://www.econt.com)
или Еконт приложението). **Не е нужен API достъп, договор или пароли в системата.**

**Стъпки:**
1. **Доставка → Еконт** → избери **„Ръчно“**.
2. **Доставка → Методи на доставка** → включи „До офис на Еконт“ и/или „До адрес“, и задай
   цената (фиксирана сума или „безплатна над …“).
3. Готово. Клиентите вече виждат Еконт на касата.

**Как изпращаш:**
- Всяка сутрин в **07:00** получаваш имейл „Доставки за днес“ с раздел **„Еконт — за изпращане“**
  (клиент + офис или адрес).
- Същият списък го виждаш и в админ панела по всяко време.
- Носиш пратките до Еконт, създаваш товарителниците в твоя Еконт акаунт/офис както обикновено.

> Цената, която клиентът плаща, е твоята фиксирана цена — тя може да се различава от реалната
> цена на куриера. Задай я така, че да ти излиза.

---

## 2. Автоматично (Еконт прави всичко вместо теб)

Системата смята реалната цена на Еконт на касата, създава товарителницата автоматично и следи
статуса — всичко от админ панела. За това трябва **API достъп** до Еконт.

**Какво ти трябва: Еконт акаунт с API достъп.**
- Това е **бизнес акаунт с договор** с Еконт (обикновен/личен акаунт няма API).
- Обади се на Еконт или влез в профила си и поискай **данни за API заявки** (потребител + парола
  за интеграция). Това е различно от паролата, с която влизаш в econt.com.
- Понякога отнема няколко работни дни, докато Еконт ги издаде.

**Стъпки:**
1. **Доставка → Еконт** → избери **„Автоматично“**.
2. Въведи **Потребител** и **Парола** за Еконт (тези за API заявки) → **„Провери връзката“**.
3. Попълни **профил на подател** (име, телефон, град на фермата и офис/адрес, откъдето тръгват
   пратките) — попълва се автоматично във всяка товарителница.
4. Задай **пакет по подразбиране** (тегло/размери) и дали да има **наложен платеж**.
5. **Доставка → Методи на доставка** → включи методите.

**Как изпращаш:**
- В „Товарителници“ натискаш иконата камионче до поръчката → генерира се номер за проследяване.
- Принтираш етикета (A4/A6), залепяш го и предаваш пратката на Еконт.
- Статусът се обновява автоматично.

---

## Кое да избера?

| | Ръчно | Автоматично |
| --- | --- | --- |
| Нужен Еконт API / договор | **Не** | Да |
| Цена за клиента | фиксирана (ти я задаваш) | реалната на Еконт |
| Товарителница | правиш я сам | автоматично |
| Проследяване | ръчно (по желание) | автоматично |
| Колко бързо тръгваш | веднага | след като Еконт даде API |

**Започни с „Ръчно“.** Като се навиеш, поискай API от Еконт и мини на „Автоматично“ — нищо не
губиш.
